# User API Documentation

## Create User
#### URL : `/api/user/`

#### Method : `POST`

#### Data :

```json
{
    "email": String,
    "password": String
}


```

#### Response :

```json
{
    "id": "00eec4dc-45d9-43ab-bee7-5f69dd1aba8b",
    "email": "email@mail.com",
    "created_at": "2020-12-04T07:29:39.255688Z",
    "updated_at": "2020-12-04T08:36:16.047932Z"
}
```
